	example: ALogin (ActivityLogin)

2) All views should start from abbreviations (depends on type of the view) and then name of the view 
	example: EditText - etLogin
			 TextView - tvLabel
			 ListView - lvUsers 
			 etc ...

3) All interfaces begins with capital 'I'
	example: ISuperActivity

	example: Lets assume that entity represents user
			 ModelUser
